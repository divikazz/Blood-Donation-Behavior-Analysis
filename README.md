# Blood-Donation-Behavior-Analysis
This project explores blood donation behavior using historical donor data.
The analysis focuses on donation frequency, recency, and donor retention patterns.

Dataset: Blood Transfusion Service Center Dataset
Features:
- Recency (months)
- Frequency (times)
- Monetary (c.c. blood)
- Time (months)
- Target variable (donated in March 2007)
  
Tools:
Python
Pandas
Matplotlib
Jupyter Notebook

Project Structure:
data/ → dataset
notebooks/ → analysis notebook
